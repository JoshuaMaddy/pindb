# Shop Creation Guide

Shops are the vendors, stores, and studios that produce or sell pins. A shop entry is shared across every pin that references it, so getting it right benefits the entire database.

---

## Before You Create a Shop

**Search first.** Shops must be unique by name. If the shop already exists under a slightly different spelling, add an alias rather than creating a new entry.

If a shop has rebranded, update the existing entry and add the old name as an alias — do not create a second shop.

---

## What Makes a Good Shop Entry

- One entry per real-world shop. If the same person sells under two distinct brands, that warrants two entries.
- Do not create shop entries for large general retailers (Amazon) unless the pin was produced by or in direct partnership with that retailer.
- A shop with only one pin in the database is still worth entering correctly

---

## Shop Name

Use the shop's current public-facing name, exactly as they present it. Preserve capitalization. If the shop uses an ampersand, use an ampersand; if they use "and", use "and".

When in doubt, use the name displayed on their storefront or social media header.

## Shop Description (optional)

A brief, factual description of the shop. Focus on what they make or sell, any notable specialties, and the country or region they operate from if relevant. Keep it neutral and factual. You may include the shop's official blurb if it has one.

## Shop Activity

Mark a shop as **inactive** if they have permanently closed or have stopped producing pins. Do not mark a shop inactive simply because they have no current listings — many shops operate in waves.

## Shop Aliases

Alternate names the shop can be found by. Use aliases for:
- Common abbreviations
- Former names after a rebrand
- Names with different punctuation or spacing (`Pin & Co` / `Pin and Co`)

## Shop Links

External URLs associated with the shop — their Etsy page, website, Instagram, etc. Links help users find the shop's storefront directly from the database.
