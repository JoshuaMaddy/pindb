# Pin Set Creation Guide

A pin set is a named, ordered collection of pins. Sets can represent an official product (a boxed set sold together), a thematic series (all pins from a specific convention), or a curated list (a community-maintained reference collection).

---

## When to Create a Set vs. Using Tags

Tags and sets overlap in purpose but serve different needs:

- Use a **tag** when the grouping is a factual property of each individual pin (`Studio Ghibli`, `Cat`).
- Use a **set** when the grouping is about a specific release or collection, and the order or membership is meaningful (`Spirited Away 2023 Box Set`).

A pin can belong to both a tag and a set — they are not mutually exclusive.

---

## Types of Sets

### Global sets

Global sets have no owner and are maintained by editors and administrators. They appear in the database as curated collections. Use global sets for:
- Official product sets (a manufacturer's boxed set)
- Series or event-based groupings (all pins from a specific convention year)
- Thematic reference collections that the community would find useful

Submitting a global set requires direct admin approval, and currently requires speaking directly with an admin. 

### Personal sets

Personal sets are owned by a specific user and appear on their profile. You manage your own personal sets from your profile — this guide focuses on global sets, which go through the editor workflow.

---

## Set Name

The name of the set as it is publicly known. For official product sets, use the name printed on the packaging or marketing. For community or thematic sets, use a clear, descriptive name, like `Long Cats [Aubry Joi]`.

## Set Description (optional)

Explain what the set is and why the pins belong together. For official sets, include the manufacturer, release year, and any notable context. For community sets, describe the curation criteria so others know what belongs.

## Set Pins

Pins in a set have a specific order. For official sets, use the order printed on the packaging or the numbering on the pins themselves. For thematic sets, use a logical order — chronological, alphabetical, or whatever best serves someone browsing the set.

## Set Links

External URLs related to the set — a product page, an announcement post, a convention page, etc.
